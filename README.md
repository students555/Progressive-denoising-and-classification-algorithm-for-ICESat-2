# ICESat-2 光子点云去噪与分类系统

基于 KNN-Otsu 动态阈值、自适应椭圆密度聚类与旋转 IQR 箱线图的三阶段渐进式去噪流程，附带 ATL08 风格的地面 / 植被 / 冠层分类模块，提供 PyQt5 图形界面与全流程可视化。

> 适用对象：ICESat-2 ATL03 级单光子计数激光测高数据（沿轨距离 + 高程两列即可使用），亦可用于其他结构相似的离散点云去噪场景。

## 项目简介

ICESat-2 ATL03 数据以单光子计数方式记录沿轨高程，原始点云中信号光子与背景噪声混叠严重，传统单一阈值或固定密度参数的去噪方法在复杂地形（陆地、植被覆盖区）下普适性有限。

本项目实现了一套三阶段渐进式去噪流程，并在去噪后的信号点上附加了一个基于相对地表高度的地表覆盖分类模块，用于将信号光子进一步划分为地面点、植被点与冠层点。整个流程通过 PyQt5 封装为带参数面板、进度反馈与分步骤结果导出的桌面应用程序，所有中间结果均以图表呈现，便于逐阶段核查去噪效果。

## 功能特性

- **三阶段去噪流程**：孤立噪声剔除 → 地形自适应密度聚类 → 局部旋转箱线图精细去噪，逐阶段输出统计图与中间数据。
- **参数自适应估算**：可根据点数、沿轨点间距、高程范围自动估算 K 值、滑动窗口、DP 简化阈值、地形分段间隔、椭圆长半轴等参数，亦支持在"高级设置"中手动覆盖。
- **分步结果导出**：每个结果标签页均可单独导出图像（PNG / PDF / SVG）与对应阶段的数据表（CSV / Excel）。
- **点云分类子模块**：对去噪后的信号点云执行 ATL08 风格的相对高度分类，划分地面点、植被点与冠层点，分类参数（地表估计窗口、地表低分位、植被起算高度、植被/冠层分界）均可交互调整。
- **界面自适应**：左侧参数面板带滚动区并设置最小尺寸，适配不同分辨率屏幕。

## 方法原理

| 阶段 | 方法 | 说明 |
| --- | --- | --- |
| 阶段 1 | KNN 距离 + 滑动窗口 Otsu 动态阈值 | 计算每个点到其 K 近邻的平均距离，在沿轨方向上按窗口/步长划分区间，对每个区间用 Otsu 法（Otsu, 1979）求局部分割阈值，剔除局部稀疏（孤立）点。 |
| 阶段 2 | Douglas-Peucker 地形简化 + 旋转椭圆密度聚类 | 先按等间隔统计沿轨地形直方图峰值得到信号中心点，再用 DP 算法（Douglas & Peucker, 1973）简化为分段折线并取各段倾角，以倾角旋转的椭圆邻域代替圆形邻域统计密度，按数据分布自适应估算密度阈值 MinPts，执行连通域聚类（思路上类似 DBSCAN，但邻域为方向自适应椭圆）。 |
| 阶段 3 | 分段旋转 + IQR 箱线图去噪 | 将信号点按固定长度分段，在每段内搜索使高程分布标准差最小的旋转角，旋转后用箱线图 IQR 规则剔除局部离群点，对地形坡度变化较大的区段更稳健。 |
| 分类模块 | ATL08 相对高度法 | 沿轨分窗统计低分位高程作为地表估计，计算每个信号点相对地表的高度，按高度阈值划分地面点 / 植被点 / 冠层点，方法思路参考 ATL08 陆地与植被产品（Neuenschwander & Pitts, 2019）。 |

## 运行环境

- Python 3.8 及以上
- 依赖库：PyQt5、numpy、pandas、scipy、scikit-image、matplotlib（详见 `requirements.txt`）
- 操作系统：Windows / macOS / Linux 均可运行（需有可用的 Qt 平台插件）
- 中文图表显示依赖本机已安装 SimHei 或同类中文字体，缺失时部分图表文字可能显示为方块

## 安装

```bash
git clone <repository_url>
cd <repository_directory>
pip install -r requirements.txt
```

## 快速开始

```bash
python APP.py
```

1. 点击「浏览...」选择 CSV / Excel 数据文件，并确认「距离列名」「高程列名」与文件实际列名一致（默认为 `Along-track distance` / `Elevation`）。
2. 默认勾选「启用自适应参数」，程序会按数据规模自动估算各阶段参数；如需精细控制，取消勾选后在「高级设置」中手动调整。
3. 点击「开始去噪」，依次查看「阶段1」「阶段2」「阶段3」及「总览」标签页中的图表结果。
4. 在任意标签页点击「保存图像」或「保存本阶段数据」单独导出该阶段结果。
5. 去噪完成后点击「点云分类」，在弹出窗口中调整地表估计窗口、地表低分位、植被起算高度等参数，查看地面 / 植被 / 冠层分类结果并导出。
6. 点击「保存去噪结果」导出包含 `Label` 标记列的完整数据表。

## 界面说明

- **左侧参数面板**：输入数据配置、参数模式切换、高级参数（仅在关闭自适应模式时可编辑）、处理状态与进度条、操作按钮。
- **右侧结果区**：以标签页形式展示各阶段图表，支持 Matplotlib 原生缩放、平移与导出工具栏。
- **点云分类窗口**：独立对话框，左侧为分类参数表单，右侧为分类结果散点图（含估计地表线与图例）。

## 参数说明

| 参数 | 含义 | 默认 / 自适应估算方式 |
| --- | --- | --- |
| K（KNN 邻居数） | 阶段 1 计算平均邻域距离时的近邻点数 | 随点数缓增，自适应范围 20–80 |
| 阶段1 窗口 / 步长 | Otsu 动态阈值的滑动窗口长度与移动步长（按点数索引） | 与总点数挂钩 |
| DP 简化阈值 | Douglas-Peucker 算法允许的最大垂距 | 高程范围的约 1% |
| 地形划分间隔 (m) | 阶段 2 沿轨分段统计信号中心点的间隔长度 | 约将测线分为 60 段 |
| 椭圆长半轴 a | 密度聚类邻域椭圆的长半轴（短半轴固定为 a/6） | 覆盖约 30 个相邻点的沿轨跨度 |
| 阶段3 分段长度 | 旋转箱线图去噪的分段点数 | 默认 5 |
| 阶段3 IQR 系数 | 箱线图离群点判定的 IQR 放大系数 | 默认 0.01（精细去噪） |
| 阶段3 角度步长 (°) | 局部最优旋转角搜索的角度分辨率 | 默认 2 |
| 地表估计窗口 (m) | 分类模块沿轨地表估计窗口长度 | 默认 20 |
| 地表低分位 (%) | 窗口内用作地表高程的分位数 | 默认 10 |
| 植被起算高度 (m) | 相对地表高度超过该值才计入植被/冠层 | 默认 0.5 |
| 植被/冠层分界 (m) | 区分植被点与冠层点的相对高度阈值 | 默认自动取正相对高度中位数，可手动覆盖 |

## 输出说明

- **去噪结果表**：原始数据表附加 `Label` 列，`1` 表示最终保留的信号点，`0` 表示被剔除的噪声点。
- **分类结果表**：包含原始距离 / 高程列，以及 `Ground_Elev`（估计地表高程）、`Relative_Height`（相对地表高度）、`Class_Code`（0=地面、1=植被、2=冠层）、`Class_Name`（对应中文类别名）。
- **阶段中间数据**：各标签页单独导出的 CSV / Excel，列名与原始距离 / 高程列一致。

## 已知限制

- 阶段 2 的椭圆邻域统计为逐点两两比较，复杂度近似 O(N²)，对单条测线超过数万点的数据集运行时间会明显增加，暂未做空间索引加速。
- 当前仅提供中文界面与中文图例，国际化（i18n）尚未支持。
- 暂无自动化测试，参数估算公式为经验启发式，建议在正式产出前结合「阶段1/2/3」中间图核查去噪效果。
- 分类模块假设信号点已基本不含噪声，若去噪不充分会直接影响地表估计与分类结果。
